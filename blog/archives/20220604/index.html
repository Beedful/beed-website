RETURN TO BEED

Do you hate needing to use your hands to calculate math? Well, you don't need to, now!
You can just input the numbers and the method (addition, subtraction, multiplication etc), it will return the answer to you!

In this blog, I'll teach you how to make addition, subtraction, multiplication and division functions and display the result.
First, we need to handle user input. We can use the input() built-in function for this.
Keep in mind that you need at least a bit of knowledge of Python for this. Please make sure that you have Python installed in your computer.

Step 1, take user input
Like I said, you can use the input() function for this. Let's create 2 variables.

num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))

Alright, we created the variables.

Step 2, choose method
Choosing method is fairly easy, you just use the input() function again.

print("Choose your method.")
print("[1] Addition")
print("[2] Subtraction")
print("[3] Multiplication")
print("[4] Division")
method = input("Choose: ")

Now, how this works is that it prints out the options and asks you to select one.

Step 3, addition, subtraction, multiplication, division
This is super easy. You just need to create four functions and return a value.

def add(a, b): return a + b
def sub(a, b): return a - b
def mul(a, b): return a * b
def div(a, b): return a / b

Now that you have finished making the functions, it's time to input it in your code.
Just after the method = input("Choose: ") line of code, add this:

if method == "1": print(add(num1, num2))
elif method == "2": print(sub(num1, num2)))
elif method == "3":print(mul(num1, num2))
elif method == "4": print(div(num1, num2))
else: print("That choice is not valid!")

Ok, you've done it!

Whole code

def sub(a, b): return a - b
def mul(a, b): return a * b
